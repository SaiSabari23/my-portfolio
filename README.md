Personal Portfolio Website

A modern and interactive personal portfolio website designed to showcase my education, skills, projects, certifications, and contact details.
The portfolio features glassmorphism design, smooth animations, and responsive layout to provide a professional and engaging user experience.

🚀 Project Overview

This portfolio website highlights my technical skills, academic background, and personal projects.
It includes interactive UI elements, animated sections, and modal popups to display certificates and resume details.

The website is designed to work smoothly across desktop and mobile devices.

✨ Features

Modern glassmorphism UI design with glowing hover effects

Smooth scroll animations and typing text animation

Interactive project and certification cards

Modal popup to view certificates and resume

Responsive layout for mobile, tablet, and desktop

Direct GitHub, LinkedIn, and email contact links
